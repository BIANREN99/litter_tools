# 英语单词学习工具使用说明书

## 1. 工具简介

英语单词学习工具是一款专为英语学习者设计的高效学习助手，支持单词导入、中英文互译练习、学习进度管理和Word文档导出等功能。通过直观的界面和智能的学习算法，帮助用户快速掌握英语单词。

![工具界面预览](https://img.icons8.com/fluency/128/000000/learning.png)

## 2. 系统要求

- 现代Web浏览器（Chrome 80+、Firefox 75+、Safari 13+、Edge 80+）
- 支持HTML5、CSS3和JavaScript
- 建议使用桌面设备获得最佳体验

## 3. 功能概述

### 3.1 单词导入

- 支持TXT格式（英文:中文）
- 支持CSV格式（英文,中文）
- 提供示例文件快速开始

### 3.2 学习模式

- 中文 → 英文：根据中文意思写出对应的英文单词
- 英文 → 中文：根据英文单词写出对应的中文意思

### 3.3 显示模式

- 单个单词：逐个单词进行练习
- 单词列表：查看所有单词并选择特定单词练习

### 3.4 学习进度管理

- 自动记录已学习单词
- 统计正确和错误次数
- 计算学习进度和正确率

### 3.5 Word文档导出

- 随机选择50个英文单词
- 生成适合默写的Word文档
- 格式：英文单词:______

## 4. 快速开始

### 4.1 使用示例文件

1. 点击"使用示例 TXT"或"使用示例 CSV"按钮
2. 系统自动加载示例单词列表
3. 选择学习模式和显示模式开始学习

### 4.2 导入自定义单词

1. 准备符合格式要求的TXT或CSV文件
2. 点击"拖放文件或点击选择"区域
3. 选择本地文件或直接拖放文件到指定区域
4. 等待系统解析完成后开始学习

## 5. 详细操作指南

### 5.1 单词导入详解

#### TXT文件格式

TXT文件需遵循以下格式：
```
hello:你好
world:世界
example:例子
```

每行一个单词，英文和中文之间用冒号（:）分隔。

#### CSV文件格式

CSV文件需遵循以下格式：
```
hello,你好
world,世界
example,例子
```

每行一个单词，英文和中文之间用逗号（,）分隔。

### 5.2 学习模式操作

#### 中文 → 英文模式

1. 选择"中文 → 英文"学习模式
2. 系统显示中文意思
3. 在输入框中输入对应的英文单词
4. 点击"检查答案"按钮验证
5. 查看反馈后点击"下一个单词"继续

#### 英文 → 中文模式

1. 选择"英文 → 中文"学习模式
2. 系统显示英文单词
3. 在输入框中输入对应的中文意思
4. 点击"检查答案"按钮验证
5. 查看反馈后点击"下一个单词"继续

### 5.3 显示模式切换

#### 单个单词模式

- 适合专注学习，逐个突破单词
- 提供即时反馈和进度跟踪
- 自动记录学习历史

#### 单词列表模式

- 概览所有单词
- 支持搜索功能快速定位单词
- 显示每个单词的学习状态
- 可选择特定单词进行练习

### 5.4 学习进度查看

- 进度条显示总体学习进度
- 统计面板显示已学单词数、正确次数和错误次数
- 单词列表中显示每个单词的掌握程度

### 5.5 Word文档导出

1. 点击顶部的"导出Word"按钮
2. 系统自动随机选择50个英文单词
3. 生成符合默写格式的Word文档
4. 浏览器自动下载生成的文档

## 6. 学习技巧

### 6.1 高效学习建议

- 每天固定时间学习，保持规律
- 每次学习20-30分钟，避免疲劳
- 先使用"单词列表"浏览所有单词，再用"单个单词"模式练习
- 重点关注错误率较高的单词

### 6.2 复习策略

- 第一次错误：标记为需要复习
- 第二次错误：加强记忆，多写几遍
- 第三次错误：查找相关例句，理解单词用法
- 隔天后再次复习已学单词

## 7. 常见问题解答

### 7.1 导入问题

**Q: 为什么我的文件无法导入？**
A: 请检查文件格式是否正确，TXT文件使用冒号分隔，CSV文件使用逗号分隔。确保文件编码为UTF-8。

**Q: 导入时提示"文件解析错误"怎么办？**
A: 可能是文件格式不符合要求，请检查文件内容，确保每行只有一个单词对，且分隔符使用正确。

### 7.2 学习问题

**Q: 如何重置学习进度？**
A: 点击顶部的"重置进度"按钮，系统将清除所有学习记录。

**Q: 练习时如何跳转到特定单词？**
A: 切换到"单词列表"模式，点击对应单词的"练习"按钮。

### 7.3 导出问题

**Q: 导出的Word文档为什么只有50个单词？**
A: 为了便于默写练习，系统默认随机选择50个单词。如果需要更多单词，可以多次导出或手动编辑文档。

**Q: 导出的文档在某些设备上显示异常怎么办？**
A: 请确保使用兼容的Word版本打开文档，或尝试使用其他文档编辑软件。
