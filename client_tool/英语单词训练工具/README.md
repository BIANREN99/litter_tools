# 英语单词学习套件软件说明书

## 软件概述

"英语单词学习套件"
是一款集成单词格式转换与单词学习管理的多功能工具软件，专为英语学习者和教育工作者设计。软件采用直观的图形界面，支持多格式文件处理、个性化学习模式和批量导出功能，帮助用户高效管理单词资源、提升学习效率。

## 运行环境

-   操作系统：Windows 7/8/10/11、macOS 10.12+、Linux
-   依赖环境：Python 3.6+
-   必要库：PyQt5、python-docx、csv、re、pathlib

## 功能模块详解

# 一、单词格式转换器（word_processor_gui.py）

## 1. 核心功能

-   将输入的单词文件（TXT/CSV/Markdown）批量转换为
    Markdown、CSV、JSON、TXT 四种格式
-   自定义输出目录
-   转换过程可视化显示

## 2. 使用步骤

### 2.1 文件选择

-   点击"浏览文件"选择文件（支持 .txt、.csv、.md）
-   显示文件名或"未选择文件"
-   输出路径默认为 ./output，可修改

### 2.2 格式选择

默认全选，可勾选： - Markdown - CSV - JSON - TXT

### 2.3 启动转换

-   点击"开始转换"
-   显示进度条与提示
-   完成后弹窗提示保存路径

### 2.4 转换规则

-   加载文件占 10%
-   创建目录 20%
-   各格式均分剩余 70%
-   输出文件为 words.xxx

# 二、英语单词学习工具（English Word Learning Suite.py）

## 2.1 单词学习标签页

### 核心功能

-   英译中 / 中译英
-   答案检查、即时反馈
-   学习进度统计

### 使用步骤

1.  加载单词文件（txt/csv/md）
2.  选择学习模式
3.  下一个 → 输入答案 → 检查答案
4.  统计显示总数/已学/剩余

## 2.2 导出 Word 标签页

### 核心功能

-   批量生成默写卷 + 答案卷
-   多套生成与单词去重
-   模式独立记录

### 使用步骤

-   选择模式、单词数量、套数
-   选择导出路径
-   点击导出 Word 文档

### 导出规则

-   两模式已导出集合独立
-   优先导出未使用单词
-   Word 文档包含标题、说明、表格、答案页

# 支持的文件格式规范

## TXT

支持：\|、,、:、制表符、空格

## CSV

至少 3 列，自动跳过标题

## Markdown

仅读取表格格式

# 输出格式

-   转换器：md、csv、json、txt
-   学习工具：docx
